# ERP Management System

## Overview
This is a full-stack ERP system built using MERN stack.

## Features
- Authentication (JWT)
- Products, Customers, Suppliers
- Sales Orders & Purchase Orders
- GRN (Inventory Update)
- Invoices
- Dashboard

## Tech Stack
- React
- Node.js
- Express
- MongoDB

## Run Project

### Backend
cd backend
npm install
node server.js

### Frontend
cd frontend
npm install
npm start
